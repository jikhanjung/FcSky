# 20260616_047 — 녹아웃 대진표 비주얼화 + 대회 상세 부문 탭

## 한 일

### 대진표 비주얼 (준결승 → 결승 연결선 + 결승 중앙 정렬)
- 대회 상세의 녹아웃 대진표를 토너먼트 형태로 재작성.
- 구조: 준결승 칸(우리 준결승 + 반대편 준결승) 2개 → 결승 1개.
- CSS 연결선(`├` 형태): 각 준결승에서 가로선 → 세로선으로 합류 → 결승으로 가로선.
  결승은 두 준결승 사이(세로 중앙)에 위치. flex `align-items:stretch` + 셀 `flex:1`로
  카드 높이에 무관하게 중앙 정렬·연결선 정렬 유지. 칸 제목은 absolute로 빼 정렬 보존.
- 승자 굵게, 결승 카드 강조 테두리. 가로 스크롤 대응.

### 대회 상세 부문 탭
- 부문이 있으면 **부문별 Bootstrap 탭**으로 구성. 각 탭에 그 부문의 출전팀·녹아웃 대진표·
  조별리그 경기만 표시. 부문 없는 대회(리그)는 탭 없이 단일 패널.
- 부문 미지정 경기/출전이 있으면 '기타' 탭. 패널 내용은 `_competition_panel.html` 부분
  템플릿으로 분리.
- 뷰: `competition_detail`이 `panels`(부문별 {출전팀·대진표·조별리그}) 생성.
  `_bracket_for()` 헬퍼로 부문별 준결승/결승 구성.
