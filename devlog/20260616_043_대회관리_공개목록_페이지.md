# 20260616_043 — 대회 목록·상세(공개) + 대회 관리(staff) 페이지

## 한 일

대회 조회/관리 화면을 신설했다. **목록·상세 조회는 누구나**, 생성·수정·삭제는 staff.

### 공개 (누구나)
- `competitions/` `competition_list` — 모든 대회를 연도별로 묶어 카드 표시(종류·주최·부문·경기 수).
- `competitions/<slug>/` `competition_detail` — 대회 정보 + 부문 + 출전팀 + 경기 결과 + 순위표 링크.
- `Competition.get_absolute_url` 추가(→ 상세).

### 관리 (staff)
- `manage/competitions/` `competition_manage` — 목록(연도·종류·부문·경기 수) + 수정/삭제 + 새 대회.
- `manage/competitions/add/`·`<slug>/edit/` — `CompetitionForm`(name/slug/kind/year/organizer/
  description) + **부문 체크박스**(20-30대/40대/50대/오픈)로 Division 생성·삭제 동기화. 슬러그는
  비우면 자동(한글→ASCII 폴백 + 중복 회피).
- `<slug>/delete/` — 삭제. 연결된 경기가 있으면(`Match.competition` PROTECT) 차단하고 안내.

### 네비
- "대회" 드롭다운 **맨 끝에 "모든 대회"** → 공개 목록.
- "관리" 드롭다운에 **"대회 관리"** → 관리 목록.
