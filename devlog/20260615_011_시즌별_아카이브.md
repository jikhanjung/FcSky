# 20260615_011 시즌별 아카이브 화면

Phase 3 마지막 항목인 **시즌별 아카이브**를 구현했다. 시즌을 골라 해당 시즌의
성과를 한곳에서 보는 허브 페이지.

## 추가/변경

- **`apps/matches/services.py` (신규)** — 집계 헬퍼 추출.
  - `finished_matches(season)` · `club_record(matches)` · `our_events(season)` ·
    `event_ranking(events, type, limit)`
  - 기존 `stats()` 뷰의 인라인 집계를 이 헬퍼로 교체(중복 제거). 아카이브 뷰와 공유.
- **`apps/competitions/views.py`**
  - `season_index` — 시즌 목록(종료 경기 수·입상 수 annotate).
  - `season_detail` — 클럽 요약 + 팀별 전적 + 출전 대회 + 입상 + 득점/도움 TOP +
    경기 결과. 순위표/일정/득점순위/명예의전당으로 시즌 필터 deep-link.
- **URL** — `/seasons/`, `/seasons/<int:pk>/` (pk 기반: `Season.year`는 unique 아님).
- **템플릿** — `competitions/season_index.html`, `season_detail.html`.
- **네비** — "아카이브" 메뉴 추가(명예의 전당 옆).
- **TODOs.md** — Phase 3 완료 표기.

## 설계 메모

- 새 집계를 만들지 않고 **기존 시즌 필터 로직을 허브로 묶는** 방향. 상세 통계·전체
  경기·조별 순위는 기존 페이지로 deep-link해 중복 렌더를 피함.
- `scorers()` 뷰는 팀·대회 필터와 `goals` 키를 쓰는 별도 구조라 이번 리팩터 대상에서
  제외(추후 통합 여지).
